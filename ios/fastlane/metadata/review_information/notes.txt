FreeDome Sphere - это профессиональный редактор для создания иммерсивного контента для купольных систем и планетариев.

Основные функции приложения:
1. Импорт и редактирование 3D контента из Unreal Engine и Blender
2. Создание купольных проекций с квантовыми эффектами
3. Интеграция с системой пространственного аудио anAntaSound
4. Поддержка комиксов в форматах .comics и .boranko
5. AIBASIC IDE для программирования интерактивных сцен
6. Экспорт проектов в формат .mbp для мобильных приложений

Приложение предназначено для профессионалов в области создания контента для планетариев, образовательных учреждений, музеев и развлекательных центров.

Для тестирования доступны демонстрационные проекты в папке samples. Приложение не требует регистрации или входа в систему для базового функционала.
